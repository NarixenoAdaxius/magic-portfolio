## Overview

The Alpha Event Dashboard is a comprehensive event management system designed specifically for ALPHA ICT Week 2024. This robust platform serves as the central hub for organizing, monitoring, and managing both esports and academic competitions during the organization's flagship event. The dashboard provides real-time participant tracking, event coordination, and administrative controls for seamless event execution.

## Key Features

- **Real-time Event Overview**: Comprehensive dashboard displaying live participant counts, event progress, and key metrics for both esports and academic categories.
- **Dual Event Categories**: Separate management interfaces for esports competitions (Call of Duty, Mobile Legends, Chess, Tekken 7, Valorant, League of Legends, Wild Rift) and academic challenges (Application Development, Beyond The Lens, Web Development, 3D Character Modeling, Game Development).
- **Participant Management**: Detailed participant tracking with team names, member information, contact details, course sections, and registration timestamps.
- **Secure Authentication**: Protected admin access with secure login system ensuring only authorized personnel can manage event data.
- **Live Count Monitoring**: Real-time participant count updates with automatic refresh capabilities for accurate event monitoring.
- **Data Export & Analytics**: Comprehensive participant data management with sorting, filtering, and export capabilities for post-event analysis.

## Technologies Used

- **Next.js**: For fast, server-rendered React applications with optimal performance and SEO.
- **MongoDB**: For storing and managing participant data, team registrations, and event information.
- **ShadCN**: For building consistent, accessible, and reusable UI components.
- **Tailwind CSS**: For rapid styling and responsive design with utility-first CSS framework.
- **Vercel**: For seamless deployment and hosting with excellent performance optimization.

## Challenges and Learnings

One of the primary challenges was creating a dashboard that could handle real-time data updates for multiple concurrent events while maintaining system performance. Managing participant registrations across various competition categories required careful database design and efficient query optimization.

Implementing secure authentication was crucial given the sensitive nature of participant data and event management. We developed a robust admin authentication system that ensures only authorized personnel can access and modify event information while maintaining user-friendly navigation.

Another significant challenge was designing an interface that could effectively display large amounts of participant data in an organized, searchable format. The table-based participant management system required careful consideration of data visualization, filtering capabilities, and responsive design to ensure usability across different devices.

Real-time count updates presented technical challenges in maintaining accuracy while preventing database overload. We implemented efficient polling mechanisms and optimized database queries to ensure live counts remain accurate without compromising system performance.

## Outcome

The Alpha Event Dashboard successfully managed ALPHA ICT Week 2024, handling hundreds of participants across multiple competition categories. The system provided organizers with real-time insights into event participation, enabling quick decision-making and efficient event coordination.

The dashboard streamlined the registration process and eliminated manual participant tracking, reducing administrative overhead and minimizing errors. The secure authentication system ensured data integrity while the intuitive interface allowed multiple administrators to collaborate effectively during the event.

The success of this dashboard system has established it as the standard event management platform for future ALPHA activities. The scalable architecture and comprehensive feature set provide a solid foundation for managing larger events and expanding functionality based on organizational needs.

Post-event analytics capabilities have provided valuable insights for improving future events, while the participant data management system has enhanced the organization's ability to maintain member engagement and communication beyond individual events.
